In this work you can find these commands, arranged in numerical order:

0-Script that prints the path name of the current working directory.

1-Script that shows the list of contents of the current directory.

2-Script that changes the working directory to the user's home directory.

3-Script that displays the contents of the current directory in long format.

4-Script that displays the contents of the current directory, including long-form and hidden files.

5-Script that displays the contents of the current directory, including long-form and hidden files. Regardless of the order after ls -.

6-Script that creates a directory in another directory.

7-Script that moves a file from one directory to another.

8-Script that removes a file from a directory.

9-Script that removes a directory from another directory.

10-Script that changes the working directory to the previous one.

11-Script that lists all files in the current directory, the directory containing it, and the /boot directory including hidden files, in long format.

12-Script that when executed shows the type of the iamafile file found in the /tmp directory.

13-Script that creates a symbolic link to /bin/ls, named __ls__.

14- Script that copies all the HTML files from the current working directory to the directory that contains it. It only copies missing or outdated files.

15-Script that moves all files beginning with a capital letter to the /tmp/u directory.

16-Script that removes all files in the current working directory that ends with the character ~.

17-Script that creates the directories welcome/, welcome/to/ and welcome/to/school in the current directory.
